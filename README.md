# 每日头条
一款用Flutter写的全网热门资讯APP。支持频道订阅，换肤，无广告。自己用了一段时间，基本满足日常使用。 理论上支持 Android，iOS，但目前只支持 Android，因为 iOS 开发者账号没交钱。
[下载地址](https://github.com/CodeNeverStops/HotNewsApp/releases)
